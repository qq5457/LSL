# 灵兽录 (LingShou Lu) — Spirit Beast Record

> 挂机放置类宠物收集养成游戏 | Idle/Incremental Pet Collection RPG

**版本:** v0.9+ (朝 v1.0 迭代中)
**架构:** HTML + CSS + JavaScript（纯前端，零依赖）
**运行:** 浏览器直接打开 `index.html` 即可游玩

---

## 🎮 游戏简介

在《灵兽录》的世界中，你将成为一名御兽师，探索不同地图，遭遇并捕捉各种灵兽。游戏为**挂机放置**模式——选好地图和主力灵兽后，自动战斗、自动捕捉、自动成长。

### 核心循环

```
选择地图 → 开始探索 → 自动遇敌 → 自动战斗
                                    ↓
                    战斗胜利 → 获得经验/金币/装备
                    发现灵兽 → 自动捕捉或炼化
                                    ↓
                    培养主力 → 挑战更强地图
```

## 🐾 灵兽系统

### 品阶

| 品阶 | 颜色 | 成长系数 | 捕捉概率 |
|:----|:----|:--------|:--------|
| 普通 (Normal) | ⬜ 白 | ×1.0 | 40% |
| 精英 (Elite) | 🔵 蓝 | ×1.5 | 15% |
| Boss (Boss) | 🩷 粉 | ×2.2 | 5% |
| 宝宝 (Baby) | ❤️ 红 | ×2.4 | 0.05%（稀有） |

### 属性

五大属性：**金 → 木 → 土 → 水 → 火 → 金**（克制链）

- 克制时伤害 ×1.5
- 被克制时伤害 ×0.75

### 技能

- **输出**：撕裂、重击、穿刺、噬咬、横扫
- **防御**：护盾
- **回复**：治疗
- **增益**：狂暴

---

## ⚔️ 战斗系统

- 自动回合制，基于速度决定先手
- 技能 35% 概率触发
- 战斗最多 25 回合
- 胜利后获得：经验、金币、装备材料（10%）、装备（2%）、兽魂
- 失败后获得少量收益

---

## 🎒 装备系统

| 部位 | 加成属性 |
|:----|:--------|
| 武器 (Weapon) | 攻击 ATK |
| 护甲 (Armor) | 生命 HP + 防御 DEF |
| 饰品 (Accessory) | 速度 SPD |

**品质：** 粗制 (2-4%) → 精良 (4-7%) → 稀有 (7-12%)

背包中绿色高亮 = 比当前装备好。

## 💾 存档

- 自动存档（每 20 秒）
- 浏览器关闭时自动保存
- 支持导出/导入 JSON 文件
- 离线收益：离线的 50%（最多 1800 tick）

## 📁 项目结构

```
LSL/
├── index.html              # 主入口（原 灵兽录.html）
├── README.md               # 本文件
├── styles/
│   └── main.css            # 完整样式表（暗色古风主题）

│   ├── ui.js               # 全部界面渲染与事件绑定

├── docs/
│   ├── 大纲规划粗略版.md    # 版本路线规划

└── data/                   # (预留) 配置 JSON
```
