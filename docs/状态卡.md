【项目名称】灵兽录（挂机放置类宠物养成游戏）

【当前版本】v1.0 完成，准备进入 v2.0 第一阶段

【项目结构】
灵兽录/
├── index.html          # 主页面
├── styles/
│   └── main.css        # 全部样式
└── scripts/
    ├── config.js       # 常量、地图、技能池、装备品质表
    ├── utils.js        # 工具函数
    ├── storage.js      # localStorage 存档读写
    ├── state.js        # 游戏状态创建/导入/兼容
    ├── pet.js          # 宠物属性计算（getBaseStats不含装备、computeStats含装备百分比加成）
    ├── combat.js       # 战斗模拟、遭遇结算、离线进度
    ├── ui.js           # 全部界面渲染与事件绑定
    └── main.js         # 入口、定时循环

【核心系统】
- 5张一阶段地图（1-150级），5张二阶段地图（60-100级，已配置未启用）
- 4种品阶：普通（白）、精英（蓝）、Boss（粉）、宝宝（红），宝宝带「宝」标识
- 装备3部位（武器/护甲/饰品），品质3档（粗制/精良/稀有），属性为百分比加成
- 战斗：宠物基础属性 + 装备百分比加成 = 总属性，技能按概率触发
- 宠物仓库/装备背包均为宫格模式，装备优于当前显示绿色高亮
- 存档用localStorage，可导出导入JSON

【当前宠物数据结构】
{ id, nickname, baseName, name, tier, isBaby, mapMult, level, exp, growth, talent, base:{hp,atk,def,spd}, skills, caughtAt, element:'none', levelCap:99, breakthroughs:0, babyFocus:null }

【当前装备数据结构】
{ id, slot, slotName, name, quality, pct:{atk,def,hp,spd}, itemLevel }
